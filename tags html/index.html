Tags HTML

As tags HTML (Hypertext Markup Language) são elementos que permitem a marcação e organização de conteúdo em um documento HTML. As tags são escritas em ângulos, com o nome da tag entre os sinais de menor e maior ("<" e ">"), e são usadas para definir a estrutura e o conteúdo de uma página web. Cada tag HTML é composta por um nome e, em alguns casos, atributos que modificam o comportamento ou a aparência do elemento. As tags podem ser divididas em dois tipos principais: as tags de conteúdo e as tags de estrutura. As tags de conteúdo são usadas para adicionar texto e mídia à página, enquanto as tags de estrutura são usadas para definir a estrutura e organização geral do documento.

Entenda a Estrutura Básica

A estrutura básica e composta por 5 elementos

  • <.!DOCTYPE>
  • <html><./html>
  • < .head><./head>
  • < .title >
  • <.body><./body>

  • O <!DOCTYPE> é uma instrução do tipo declaração que informa ao navegador qual é a versão do HTML que está sendo usada no documento. Esta instrução deve estar sempre presente no início do documento HTML.

  • <html></html> é a tag que envolve todo o conteúdo do documento HTML. É nessa tag que se inicia e termina o documento HTML.

  • <head></head> é a tag que contém informações sobre o documento, como título, links para arquivos CSS e JavaScript, meta tags e outros dados que são importantes para a indexação e exibição do conteúdo da página. O conteúdo da tag head não é exibido na página web.

  • <title></title> é a tag que define o título da página, que é exibido na aba do navegador e nos resultados de pesquisa. A tag title deve estar presente dentro da tag head.

  • <body></body> é a tag que envolve o conteúdo principal da página web, como textos, imagens, vídeos e outros elementos. É nessa tag que o conteúdo da página é exibido no navegador.

Tags de Titulo

  • <h1></h1>- Título de maior valor Hierárquico
  • <h2></h2>
  • <h3></h3>
  • <h4></h4>
  • <h5></h5>
  • <h6></h6> - Título de menor valor Hierárquico

Tags de Texto

  • <p></p>- Principal tag de texto, compõe um parágrafo

  • <span>/span>- Apesar de ter uma funcionalidade e características parecidas com os parágrafos, costumam ser utilizadas apenas para pequenas informações, como legendas de um formulário, legendas de uma imagem, etc. Também pode ser utilizada para formar um container;

  • <pre>/pre>-Tag utilizada para representar texto pré-formatado. Muito utilizada para inserir códigos;

  • <b></b>- Transforma o conteúdo em negrito;

  • <i></i>-Transforma o conteúdo em itálico;

  • <br>-Essa tag não necessita de fechamento, ela executa a função de quebra de linha.

  • <hr>-Essa tag não necessita de fechamento, ela forma uma linha horizontal.
